## Top questions

**Is Nito a model, or a gateway?**
A gateway. Nito sits in front of many models from several providers, and you choose the model per request. See [How Nito Works](/get-started/how-nito-works).

**Do I need a special SDK?**
No. Nito is OpenAI-compatible: point the OpenAI SDK (or plain HTTP) at Nito's base URL and use a Nito model ID. See [Integrations](/faq/integrations).

**Does every call run in a TEE?**
No, and that is deliberate. TEE execution is the highest tier, selected per call by adding `:confidential` to the model ID. Most calls use lighter modes. See [Privacy and TEE](/faq/privacy-and-tee).

**What does Nito keep about my prompts?**
By default, metadata only. Prompt and completion contents are treated as request-local and are not persisted by the gateway. See [Zero Data Retention](/privacy/zero-data-retention).
