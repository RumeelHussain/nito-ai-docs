The Resources tab collects practical guides for building well on Nito — the material you reach for once you are past the basics. It is not tutorial material; for that, start with [How Nito Works](/get-started/how-nito-works).

Core reference tables live with the topics they belong to: **[Pricing](/models/pricing)** sits with Models, and **[errors & rate limits](/api-reference/errors-and-rate-limits/error-codes)** live in the API reference.

- [**Uptime and fallbacks**](/resources/best-practices/uptime-and-fallbacks) — stay up when a model or provider does not.
- [**Production-readiness checklist**](/resources/best-practices/production-readiness-checklist) — what to verify before you ship.
