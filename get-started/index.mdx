Nito is a private multi-LLM inference gateway. You send one kind of request to one endpoint with one key, Nito routes it to the right model, and you choose how private each call is via the model ID. If you have used the OpenAI or Anthropic API, you already know the request shapes: in most cases you only change the base URL.

This section takes you from zero to a working integration. Read the first two pages for the mental model, then pick a quickstart and make a real call.

- [**How Nito Works**](/get-started/how-nito-works) the one-key, many-models, selectable-privacy model that the rest of the docs build on.
