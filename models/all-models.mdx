This page lists every model a user can call on Nito, grouped by privacy tier. This is a **snapshot**,  `GET /v1/models` is always the source of truth, since the catalog is discovered live and changes over time.

<Note>
  Snapshot of **2026-07-18** · **41 models** (all chat). **Price /1M** is USD per 1M tokens, shown as **input · output**, read live from each model's `pricing` object. For the current list, call [`GET /v1/models`](/api-reference/models-catalog/list-models); see [Pricing](/models/pricing) for the full pricing object and cached-token rates.
</Note>

The `capabilities` object returned as a response reports what a model can do, for instance:

- `reasoning` step-by-step thinking
- `vision` image input
- `web` web search
- `code` code interpreter
- `attestable` supports a verifiable TEE attestation.

## Fetching The Live Model Catalog

Nito provides the `GET /v1/models `API endpoint to fetch the latest list of live model catalog.

```bash
curl "https://nito.dev.zprotocol.org/v1/models" \
  -H "Authorization: Bearer $NITO_API_KEY"
# add ?type=all to include non-chat kinds
```
