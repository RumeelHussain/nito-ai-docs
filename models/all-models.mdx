Every model you can call on Nito, grouped by privacy tier. This is a **snapshot** — `GET /v1/models` is always the source of truth, since the catalog is discovered live and changes over time.

<Note>Snapshot of **2026-07-18** · **41 models** (all chat). **Price /1M** is USD per 1M tokens, shown as **input · output**, read live from each model's `pricing` object. For the current list, call [`GET /v1/models`](/api-reference/models-catalog/list-models); see [Pricing](/models/pricing) for the full pricing object and cached-token rates.</Note>

**Capabilities key:** `reasoning` step-by-step thinking · `vision` image input · `web` web search · `code` code interpreter · `attestable` supports a verifiable TEE attestation.

```bash
  -H "Authorization: Bearer $NITO_API_KEY"

```
