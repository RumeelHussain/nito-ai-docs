Every model you can call on Nito, grouped by privacy tier. This is a **snapshot** — `GET /v1/models` is always the source of truth, since the catalog is discovered live and changes over time.

<Note>Snapshot of **2026-07-16** · **41 models** (all chat). For the current list at any moment, call [`GET /v1/models`](/api-reference/models-catalog/list-models) (see [Model catalog](/models/model-catalog)). Prices are per **million tokens** (input / output), derived from the `pricing` object.</Note>

**Capabilities key:** `reasoning` step-by-step thinking · `vision` image input · `web` web search · `code` code interpreter · `attestable` supports a verifiable TEE attestation.

## Fetch the live catalog

```bash
curl "https://nito.dev.zprotocol.org/v1/models" \
  -H "Authorization: Bearer $NITO_API_KEY"
# add ?type=all to include non-chat kinds
```
