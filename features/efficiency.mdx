Two features help you run leaner on Nito — trim the cost and latency of repeated prompts, and retry failed calls safely.

## In this section

- [**Prompt caching**](/features/prompt-caching) — reuse the cost and latency of a repeated prompt prefix instead of paying for it on every call.
- [**Safe retries**](/features/safe-retries) — retry a failed request safely; failed calls are not billed, so a retry only pays for the attempt that completes.
